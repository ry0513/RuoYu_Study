## 鼠标事件

| 操作                  | 方法       |
| --------------------- | ---------- |
| 点击                  | click      |
| 双击                  | dblclick   |
| 按下按键              | mousedown  |
| 松开按键              | mouseup    |
| 移入元素(不支持冒泡)  | mouseenter |
| 移出元素 (不支持冒泡) | mouseleave |
| 移入元素 (支持冒泡)   | mouseover  |
| 移出元素 (支持冒泡)   | mouseout   |
| 移动                  | mousemove  |

## 键盘事件

| 操作           | 方法     |
| -------------- | -------- |
| 按下并松开按键 | keypress |
| 按下按键       | keydown  |
| 松开按键       | keyup    |
